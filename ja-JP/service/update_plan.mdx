## 継続的に更新中...

私たちはオープンなエコシステムの構築に取り組み、コミュニティからのフィードバックに基づいて、さらなる革新的機能の探索を続けます。<Link href="/submit">あなたのアイデアやコンテンツを創造所に投稿して、より多くのユーザーに届けましょう</Link>。

### 株価データ（リスト）

_2025.8.8 リリース済み_
注目銘柄の株価をリアルタイムで追跡。投資情報をひと目で確認できます。

<ul>
	<li>最大 4 銘柄を表示</li>
	<li>最新価格</li>
	<li>騰落率の表示</li>
</ul>

</Timeline>

<Timeline>

### TickTick（滴答清单）

_2025.7.17 リリース済み_

TickTick（収集箱以外）の ToDo を Dot. デバイスへシームレスに同期。アプリで TickTick アカウントを連携すればすぐに使えます。

<ul>
	<li>指定リストの ToDo を同期</li>
	<li>デバイス側でタスク状態を管理</li>
</ul>

</Timeline>

<Timeline>

### テキスト API / 画像 API

_2025.7.17 リリース済み_

開発者向けに強力な API を提供。API キーで標準リクエストを構築し、デバイスの遠隔制御やカスタムコンテンツのプッシュ表示を実現可能にします。

<ul>
	<li>標準 RESTful API を提供</li>
	<li>カスタムコンテンツのプッシュと表示</li>
</ul>

</Timeline>
