A Model Context Protocol (MCP) server that provides tools to interact with Dot devices. It implements the official Dot API endpoints to render text and images on Dot device screens.

<CustomCloudImage osSrc="dot/docs/stvlynn_quote0-mcp_1757593061861.jpeg" alt="Runtime preview" />

### Features

-   Simple Text API: display basic text content on MindReset devices
-   Rich Text API: display text with signatures and formatting
-   Text with Icon: display text with a custom icon and a tappable link
-   Image API: display PNG images with customizable borders
