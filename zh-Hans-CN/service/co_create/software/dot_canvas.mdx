DotCanvas 是一个轻量级的可视化 Dot.编辑工具。

提供读取、创建、更新和渲染画布的接口，配套的 Web 界面允许你直接在浏览器中编辑画布元数据、视图构建函数代码，并实时预览生成的图片。

### 功能

-   在浏览器内完成所有操作，包括编辑画布，设定推送任务。
-   所有元素都可以通过 python 脚本自由动态的生成内容。
-   实时预览你的作品！
