Designed for Dot e-paper displays, this project uses QWeather 30-day weather data to generate a calendar chart and push it to Quote/0.

/>

Note: It also supports displaying a ToDo list. This repo includes an implementation for the DingTalk Calendar API to fetch schedule information. You can also use other solutions to supplement this data.

### Features

-   Display the local 30-day weather data in a calendar view
-   Show todayâ€™s local weather, minimum temperature, and maximum temperature
-   Show local weather alerts
-   Show local rain, snow, and fog conditions within the next 2 hours, or the nearest day with such conditions within the next two weeks
-   Display a ToDo list (refer to the demo to integrate DingTalk schedules, or implement your own ToDo text ingestion)

## Technical Requirements

You need some development skills and your own server (or a local machine).

Runtime environment: PHP + Apache/Nginx  
Required PHP extensions: curl, gd, mbstring
