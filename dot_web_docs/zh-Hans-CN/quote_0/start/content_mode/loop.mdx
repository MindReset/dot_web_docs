## 循环内容

循环内容模式按照用户设定的顺序和间隔时间，依次显示列表中的各项内容，类似于播放列表的循环播放机制。

<Callout type="info" title="运行机制与更新间隔">
循环内容的运行机制如下：

- 设备按照循环列表中的顺序依次显示内容；
- 在 App 的循环列表中，可通过长按拖拽调整内容的排列顺序；
- 点击内容右侧的「更多」选项，可立即在设备上显示该内容；
- 在 App 的循环列表上方，点击「调整间隔时间」可修改全局的内容切换间隔。

**续航与更新频率：**
设备使用电池供电时，较长的间隔时间可延长设备的睡眠时间，从而提升续航时长，但会降低内容更新的时效性。设备连接电源时，循环间隔最短可设置为 **1 分钟**，以实现最高频率的内容更新。
</Callout>
