## 持续更新...

我们致力于构建一个开放的生态系统，并将根据社区反馈，持续探索更多创新功能。<Link href="/submit">贡献你的想法或将你的内容加入到我们的内容工坊，触达更多用户</Link>。

### 规划中

- 笔记类内容集成：每日回顾笔记，让灵感回顾更便捷。
- Docker Images Metadata：便捷获取镜像相关数据。
- iCal 日历集成：时刻查看你的日程安排。
- 节假日安排：随时查看未来一段时间的节假安排。
- 加密货币：实时查看价格与涨跌。

### 已上线

#### 2025.12.19

- 开发者平台 API：更精细的设备控制（列出设备、查看状态、切换到下一个内容等）。<Link href="/docs/service/open">查看 API 文档入口</Link>。
- 日期书摘：根据今天的日期，随机展示一句书摘。
- 轻牛健康：绑定您的轻牛健康账号，获取体重、运动、热量摄取与消耗等健康数据。<Link href="/docs/service/studio/qnniu">查看使用说明</Link>。

#### 2025.12.05

- GitHub 用户活动：展示用户的最新活动动态，涵盖提交、拉取请求、问题等多种类型。
- GitHub 仓库数据：展示指定仓库的关键数据指标，如星标数、分支数、Fork 数等。
- Github 仓库 Pull Request：展示指定仓库的最新拉取请求动态。
- Github 仓库 Issue：展示指定仓库的最新问题动态。
- 哔哩哔哩 UP 主数据面板：展示粉丝数、关注数、投稿数等数据。
- 哔哩哔哩 UP 主投稿数据面板：展示投稿相关的数据，助力内容创作者追踪创作表现。
- Readwise 每日回顾：每日推送精选的高亮内容，帮助你知识积累与复习。
- 天气预报：多日天气预报与详情。
- Readhub 科技资讯：每日科技资讯速递，助你紧跟科技前沿动态。
- 黄金数据：每日黄金价格，助你把握投资良机。
- 天气建议：多种天气建议，助你更好规划日常生活。
- 云音乐榜单：随机获取一首音乐，碰一碰即可播放。
- 二维码：展现一个二维码，支持 URL、文本等多种内容形式。
- 号码牌：展示号码牌信息，既可以填写电话，也可以填写任何数字相关的内容。
- 历史上的今天：每日历史事件回顾，增长见识。
- 答案之书：随机生成一个答案，助你决策。
- AI 资讯快报：每日 AI 领域最新资讯，助你紧跟技术前沿。
- 欧路词典：随机回顾一个你的单词。
- 今日运势：看看今天运势如何。
- 老虎机：看看运气如何吧。

#### 2025.9.25

- Telegram 频道：订阅频道最新消息，点击可查看详情。

#### 2025.9.10

- 照片：轻松将照片设置到 Dot. 设备，随时欣赏美好回忆。
- 天气灾害预警：提供及时的预警信息，保障出行安全。

#### 2025.9.03

- 降雨概率：未来一小时的降雨概率与降雨量提醒。

#### 2025.8.8

- 日历：展示农历与节假日信息。
- 通勤限行：常用通勤路线限行提醒，避免出行受阻。
- 股票数据列表：最多显示四只股票，含最新价格与涨跌幅。
- 实时天气：当前天气、体感、风向风力等一屏查看。

#### 2025.7.17

- 滴答清单：同步指定清单的待办事项，并可在设备端管理状态。
- 文本 API / 图像 API：提供标准 RESTful API，用于远程控制或推送自定义内容。

#### 2025.5.27

- YouTube：订阅频道，获取最新视频更新，碰一碰即可查看。
