Dot. Quote/0 向けの Home Assistant カスタム Lovelace カードです。デバイス操作を 1 つの可視化パネルにまとめられます。

このカードは `HA-dot-quote0` と組み合わせて利用します。統合側がデバイス接続とサービスアクションを担当し、カード側がフロントエンド UI と操作を担当します。

### 主な機能

- **E-ink プレビュー領域**: カード上部に画面プレビューを表示し、送信前のプレビュー状態も確認可能
- **デバイス状態チップ**: オンライン状態、電源状態、バッテリー、Wi-Fi、ファームウェア、レンダリング時刻
- **Next Content**: コンテンツループの次項目へワンクリックで切り替え
- **Send Text**: タイトル・メッセージ・署名をカード上で入力して送信
- **Send Image**: 296×152 PNG をアップロードし、ディザリングと枠線を選んで送信
- **デバイスセレクター**: Home Assistant で検出済みの Quote/0 デバイスを選択
- **ビジュアルエディター**: ダッシュボード UI からカード設定を編集
- **AI 画像生成**: Gemini API を任意で設定し、送信前にピクセルアートを生成可能

### HA-dot-quote0 との関係

- **依存関係**: 先に `HA-dot-quote0` を導入すると、カードでデバイス検出と操作実行が可能
- **責務分離**:
  - `HA-dot-quote0`: 統合、センサー、サービスアクション（テキスト/画像/天気/カレンダー/システム状態）
  - `HA-dot-quote0-card`: ダッシュボード操作、プレビュー、クイック送信、デバイス選択
- **推奨運用**:
  - 日常の手動操作はカード
  - 定期・条件実行は統合サービスを自動化で利用

連携する統合ページ: [/docs/service/co_create/software/ha_dot_quote0](/docs/service/co_create/software/ha_dot_quote0)

### インストール

#### HACS（推奨）

1. HACS → フロントエンド → ⋮ メニュー → カスタムリポジトリ を開く
2. `https://github.com/Never2land/HA-dot-quote0-card` をカテゴリ Dashboard で追加
3. Dot. Quote/0 Card をインストール
4. ブラウザをハードリフレッシュ

#### 手動

1. releases から `dist/dot-quote0-card.js` をダウンロード
2. Home Assistant の `config/www/` にコピー
3. 設定 → ダッシュボード → リソース で追加:
   - URL: `/local/dot-quote0-card.js`
   - Type: JavaScript Module
4. ブラウザをリフレッシュ

### 設定

まずビジュアルエディターの利用を推奨します。カード追加で Dot. Quote/0 Card を検索し、デバイスを選択して保存します。

YAML で手動設定する場合:

```yaml
type: custom:dot-quote0-card
device_id: ABCD1234ABCD
show_preview: true
show_send_text: true
show_send_image: true
gemini_api_key: AIzaSy...
```
