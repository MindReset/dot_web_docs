# Dot. 文档仓库

**Languages:** [English](README.md) | [简体中文](README.zh-Hans-CN.md) | [日本語](README.ja.md)

欢迎来到 Dot. 系列产品的官方文档仓库！本仓库包含 Dot. App 和 Dot. 系列设备的完整使用手册、API 文档、常见问题解答及安全公告等内容。

## 📚 关于本仓库

这是 [MindReset](https://www.mindreset.tech) 旗下 Dot. 品牌的文档中心，包含以下内容：

- **产品使用手册**：详细的安装、配置及使用指南
- **设备文档**：Quote/0 等设备的完整操作说明
- **开发者文档**：开放 API 与 Studio 集成指南
- **常见问题**：故障排查与问题解决方案
- **安全公告**：产品安全更新与漏洞披露
- **固件更新日志**：版本历史与更新说明

## 🤝 参与贡献

我们欢迎并鼓励社区成员参与文档的改进！如果您发现任何错误、遗漏或想要改进文档内容，欢迎通过以下方式参与：

### 提交问题

如果您发现文档中的问题或有改进建议，请[创建 Issue](../../issues)。

### 提交 Pull Request

1. Fork 本仓库
2. 创建您的特性分支 (`git checkout -b feature/improve-docs`)
3. 提交您的更改 (`git commit -m '改进：更新 Quote/0 配对说明'`)
4. 推送到分支 (`git push origin feature/improve-docs`)
5. 创建一个 Pull Request

### 贡献指南

在提交内容前，请注意：

- **保持一致性**：遵循现有文档的格式和风格
- **准确性**：确保技术信息的准确性
- **多语言**：如果可能，请同时更新所有语言版本
- **清晰表达**：使用简洁、易懂的语言
- **图片优化**：如需添加图片，请确保文件大小适中
