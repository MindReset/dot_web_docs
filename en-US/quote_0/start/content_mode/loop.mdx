## Loop Content

Loop content refers to "the duration to wait between each item in the loop":

- The device displays content according to the sequence in the loop list
- You can press and hold to reorder items in the loop list
- You can tap "More" on the right to immediately display the current content

In the App, you can tap "Adjust Interval" above the loop list to set different interval times.

When the device runs on battery, longer intervals correspondingly extend the usage time since the interval determines how long the device sleeps between updates. However, less frequent updates reduce real-time relevance. If the device is plugged into power, the loop interval can be as short as 5 minutes, maximizing real-time relevance.
