title: Dot 加密货币行情

<GithubInfo owner="Linus-Shyu" repo="dot-crypto-ticker" />

一款轻量级 Rust 应用程序，可在你的 Dot 设备上显示实时加密货币价格，让你的桌面伴侣随时掌握市场行情。

### 功能特性

- **多币种支持**：实时追踪比特币 (BTC)、以太坊 (ETH) 和泰达币 (USDT) 价格
- **优化显示**：专为 Dot 设备屏幕设计的迷你模式，清晰易读的格式呈现
- **实时更新**：24 小时价格变化指标，配合直观的趋势箭头 (↗/↘)
- **可配置间隔**：自定义刷新频率以满足你的需求 (默认：10 分钟)
- **安全配置**：基于环境变量的 API 密钥和设备凭证设置
- **24/7 后台运行**：支持作为 macOS 后台服务持续运行
- **可靠数据**：使用 Binance API 提供准确的实时加密货币市场数据

### 显示格式

应用程序以紧凑、易读的格式呈现加密货币价格，为 Dot 设备优化：

```
BTC $111,266.93 ↗+2.44%
ETH $4,039.08 ↗+1.23%
USDT $0.9997 ↗+0.01%
```

每条信息包含带千位分隔符的格式化价格、方向指示器和 24 小时涨跌幅百分比。
