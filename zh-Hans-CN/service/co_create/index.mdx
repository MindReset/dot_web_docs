<Callout title="什么是社区共创？">
	社区共创是围绕 Dot. 生态，由开发者、设计师与爱好者基于开放
	API、协议与示例持续产出的应用与集成实践；它既关注“可用可复用”的小而美方案，也鼓励以实验精神探索电子墨水屏与
	AI、自动化的全新边界。
</Callout>

在这里，你会看到社区如何把天气、日历、待办与知识库等信息无缝送达电子墨水屏，通过文本与图片 API 的组合实现更清晰的阅读体验，并借助 MCP、Webhook 与自动化工作流把桌面智能体与物理设备联动起来；这些看似分散的灵感，最终汇聚成一种轻量而高效的人机协作方式：信息在最合适的时间与场景抵达，界面保持克制与可读，创意在开放的实现路径中相互启发与演进。
