毎日の体重と運動データをダイエットのために追跡したい場合、このテンプレートを使用できます。

<CustomCloudImage osSrc="dot/docs/1_1765452182117.jpg" alt="実際の表示" />

## 機能

iPhone でこのショートカットを実行すると、以下のデータが自動的に取得されます：

- 体重
- 体脂肪率
- アクティブエネルギー
- 歩数
- 歩行距離

データは API を通じてサーバーに送信され、画面表示が自動的に更新されます。

## 技術要件

1. iPhone の「ショートカット」のインポートと変更方法を理解していること
2. 「歩数」と「距離」のデータソースを自分で調整できること
3. 最終的な API 送信インターフェースで認証トークンとデバイス SN を自分のものに変更できること

詳細については、こちらをご覧ください：[健康リマインダー](https://ros6.com/?action=show&id=282)
