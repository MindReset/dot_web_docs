## API とは？

API（Application Programming Interface、アプリケーション・プログラミング・インターフェース）は、異なるソフトウェア間で通信やデータ交換を行うための橋渡しです。簡単に言えば、API は「通訳」や「ウェイター」のような存在で、互いに異なるプログラム同士が「会話」できるようにします。

## API はどのように動くの？

<Steps>
    <Step>
        ### リクエストを送る

        「デバイスに『今日は天気がいい』と表示して」
    </Step>
    <Step>
        ### API がリクエストを処理

        API はリクエストを受け取ると、ユーザー認証とデバイス認証を行い、正当なユーザーとデバイスからの要求かを確認します。
    </Step>
    <Step>
        ### デバイスが操作を実行

        デバイスが指示を受け、画面にテキストを表示します。
    </Step>
    <Step>
        ### 結果を返す

        「操作成功。内容をデバイスに表示しました」
    </Step>

</Steps>

## Dot. API でできること

Dot. API によって、次のようなことが可能になります。

-   重要なメール受信時、件名と一部内容を自動でデバイスに表示
-   認証コード（SMS）を受け取ったら、自動でデバイスに表示
-   玄関ベルが鳴ったら、デバイスに訪問者情報を表示
-   高度にパーソナライズされたレイアウト（個人用ダッシュボードなど）を表示
-   面白い画像や絵文字を表示

## 誰が API を使える？

1. ギーク／愛好家：ノーコードツール（ショートカット、IFTTT、Zapier など）で簡単な自動化を作る
2. プログラマー：コードを書いて複雑な自動化システムを構築する
