DotCanvas は、軽量なビジュアル Dot 編集ツールです。

キャンバスの読み込み・作成・更新・レンダリング用の API を提供します。付属の Web インターフェースでは、ブラウザ上でキャンバスのメタデータやビュー構築用関数コードを直接編集でき、生成画像をリアルタイムでプレビューできます。

### 機能

-   ブラウザ内で完結。キャンバス編集やプッシュタスクの設定が可能。
-   すべての要素は Python スクリプトで動的にコンテンツ生成が可能。
-   作品をリアルタイムでプレビュー！
