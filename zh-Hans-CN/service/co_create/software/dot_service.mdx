专为 MindReset Dot. Quote/0 墨水屏设备设计的自托管 Web 控制服务。提供现代化 Web 界面和增强的 REST API，让你轻松管理设备状态、发送文字排版、上传图片，并无缝集成到自动化工作流中。

### 功能特性

- **现代化 Web 控制台**：响应式设计，适配桌面与移动端，支持深色/浅色模式切换（跟随系统）
- **设备概览**：实时查看设备在线状态、电池电量、WiFi 信号、固件版本及当前屏幕预览
- **强大的文字推送**：支持标题、正文、署名自动排版；双模式渲染——设备内置排版 API 或服务器端文字转图片（可完全自定义字体）
- **智能图片处理**：自动缩放/裁剪至 296×152 分辨率；内置 Floyd-Steinberg、Atkinson、Sierra 等多种抖动算法；支持实时预览抖动效果
- **增强型 REST API**：基于 FastAPI 构建，自动生成 Swagger/OpenAPI 文档，封装 Dot Cloud Auth V2 协议

### 核心 API

- `GET /devices` — 获取设备列表
- `POST /text` — 发送文字
- `POST /image` — 发送 Base64 图片
- `POST /image/upload` — 上传图片文件

服务启动后访问 `http://localhost:8000/docs` 可查看完整 Swagger 文档。

### 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入 DOT_API_KEY 和 DOT_DEFAULT_DEVICE_ID

# 启动服务
python run.py
```

访问 `http://localhost:8000` 进入控制台。
