Welcome to the official documentation repository for Dot. products! This repository contains comprehensive user manuals, API documentation, FAQs, and security advisories for the Dot. App and Dot. series devices.

## 📚 About This Repository

This is the documentation center for [MindReset](https://www.mindreset.tech)'s Dot. brand, including:

- **Product Manuals**: Detailed installation, configuration, and usage guides
- **Device Documentation**: Complete operation instructions for Quote/0 and other devices
- **Developer Documentation**: Open API and Studio integration guides
- **FAQs**: Troubleshooting and problem-solving solutions
- **Security Advisories**: Product security updates and vulnerability disclosures
- **Firmware Changelogs**: Version history and update notes

## 🌍 Multi-language Support

Documentation is available in the following languages:

- **简体中文** (`zh-Hans-CN/`)

## 🤝 Contributing

We welcome and encourage community contributions to improve the documentation! If you find any errors, omissions, or want to improve the content, please participate through:

### Submit Issues

If you find problems in the documentation or have suggestions for improvement, please [create an Issue](../../issues).

### Submit Pull Requests

1. Fork this repository
2. Create your feature branch (`git checkout -b feature/improve-docs`)
3. Commit your changes (`git commit -m 'docs: update Quote/0 pairing instructions'`)
4. Push to the branch (`git push origin feature/improve-docs`)
5. Create a Pull Request

### Contribution Guidelines

Before submitting content, please note:

- **Maintain Consistency**: Follow the existing documentation format and style
- **Accuracy**: Ensure technical information is accurate
- **Multi-language**: If possible, please update all language versions simultaneously
- **Clear Expression**: Use concise and understandable language
- **Image Optimization**: If adding images, ensure file sizes are reasonable
