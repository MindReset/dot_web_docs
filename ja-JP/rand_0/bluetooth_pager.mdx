## Bluetooth リモコン

Bluetooth リモコンは、Rand/0 をさまざまな端末に接続し、ページ送り、遠隔撮影、プレゼン操作に使うための機能です。

-   多くの電子書籍リーダーに接続して、ページ送りリモコンとして使用できます。
-   スマートフォンに接続して、遠隔撮影のシャッターとして使用できます。
-   コンピューターに接続して、Keynote、PowerPoint、Figma などのプレゼンリモコンとして使用できます。
-   ファームウェア 1.2.0 以降では、Apple デバイス向けの Bluetooth 暗号化検証が改善され、接続互換性が向上します。

## ボタンモード

Rand/0 Bluetooth リモコンは複数のボタン出力モードに対応しています。モードによって、上ボタンと下ボタンが接続先デバイスへ送信する Bluetooth ボタンイベントが変わります。

| モード | 利用シーン | 説明 |
| --- | --- | --- |
| 閲星曈プロトコル | 閲星曈プロトコルに対応したハードウェア端末 | 対応するハードウェア読書端末と組み合わせて使用します。 |
| PPT リモコン | Keynote、PowerPoint、Figma Slides などのプレゼン場面 | プレゼン中の前ページ / 次ページ操作に使用します。 |
| スマートフォンページ送り | ショート動画、情報フィード、上下切り替えに対応したモバイル App | スマートフォンやタブレット上で上下にコンテンツを切り替えます。 |
| カメラリモコン | システムカメラ、または音量キー撮影に対応したカメラ App | 遠隔シャッターとして使用できます。 |
| メディアコントロール | 音楽、動画、またはメディアキー操作に対応した App | 前の曲 / 次の曲などのメディア操作に使用します。 |
| 読書モード | WeRead、Apple Books、電子書籍リーダーなどの読書場面 | 読書中のページ送りに使用します。 |
| 随想编码 | AI ショートカット操作に対応したワークフローまたは App | デフォルトでは上ボタンが **AI 受け入れ**、下ボタンが **AI 呼び出し** です。 |
| カスタム | 上下ボタンの動作を個別に設定したい場面 | 上ボタンと下ボタンに異なるボタンイベントを割り当てます。 |

<Callout type="info" title="随想编码について">
随想编码モードでは、デフォルトで上ボタンが **AI 受け入れ**、下ボタンが **AI 呼び出し** に設定されます。実際の動作は、接続先のデバイスまたは App が対応するボタンイベントをサポートしているかによって異なります。
</Callout>

### カスタムボタン

カスタムモードでは、上ボタンと下ボタンのどちらにも、次のボタンイベントを割り当てることができます。

| カテゴリ | 選択できるボタンイベント |
| --- | --- |
| 操作なし | なし |
| カメラ / 音量 | 撮影 / 音量+、撮影 / 音量- |
| メディアコントロール | 再生 / 一時停止、次の曲、前の曲 |
| 一般的なキーボード | Enter、Space、Page Up、Page Down、左矢印、右矢印 |
| ページ送り制御 | ページ送り-前ページ、ページ送り-次ページ、ページ送り-戻る、ページ送り-確認、ページ送り-スリープ、ページ送り-電源オフ、ページ送り-電源オフ前 |
| プレゼン制御 | F5 開始、Esc 終了、B ブラックスクリーン |
| AI 制御 | AI 呼び出し、AI 受け入れ、AI インライン |

<Callout type="info" title="互換性について">
Bluetooth リモコンの実際の対応状況は、Rand/0 のファームウェア、接続先デバイス、そして対応する App またはハードウェアが Bluetooth ボタンイベントに対応しているかによって決まります。

App、システム、ハードウェアのファームウェア更新により、以前は対応していたものが一時的に使用できなくなる場合があります。反対に、以前は非対応だった App やデバイスが、更新後に対応する場合もあります。

対応状況の変化に気づいた場合は、ぜひお知らせください。この表を更新し、必要に応じて該当する App またはハードウェアメーカーへ連絡し、よりよい互換性対応を目指します。
</Callout>

## 操作

1. 下ボタンを長押しして入ります。
2. 端末が **接続待ち** と表示するまで待ちます。
3. モバイル端末で Bluetooth 設定を開きます。
4. **Rand/0 Pager** を見つけてペアリングします。
5. ペアリング後、端末の案内に従って操作します。
