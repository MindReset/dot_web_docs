## 什么是 RSS？

RSS（Really Simple Syndication 或 Rich Site Summary）是一种 Web 内容聚合格式，允许用户订阅网站的更新内容。通过 RSS，您可以在一个地方集中获取多个网站的最新文章、新闻或更新，无需逐一访问每个网站。

<Callout title="关于 RSSHub" type="info">

RSSHub 是一个开源的 RSS 生成器，可以为没有提供 RSS 的网站生成 RSS 订阅源。通过 RSSHub，您可以将各种网站的内容转换为 RSS 格式，方便统一订阅和管理。在 Dot. 内容工坊中，如果您看到了数据解析为「RSSHub」，则意味着该数据是基于 RSSHub 的 API 生成的。

### 相关链接

-   GitHub 地址：[DIYgod/RSSHub](https://github.com/DIYgod/RSSHub)
-   官方文档：[docs.rsshub.app](https://docs.rsshub.app)

通过 RSSHub，您可以轻松地将各种网站内容转换为 RSS 格式，实现真正的内容聚合订阅。

</Callout>

## 使用 RSS 订阅

<Steps>
<Step>
	### 检索或获取 RSS 订阅链接

    一般来说，您可以在网站的底部或侧边栏找到 RSS 订阅链接，通常以「RSS」或「订阅」字样标识。如果网站没有提供 RSS 链接，您可以使用 RSSHub 官方站点或自行搭建的 RSSHub 来生成。

    RSS 链接格式为 `https://example.com/rss` 或 `https://example.com/feed`。
