Dot 用に開発された電子ペーパー向けに、QWeather の 30 日間の天気データを用いてカレンダー形式のグラフを作成し、Quote/0 へプッシュします。

<CustomCloudImage osSrc="dot/docs/iShot_2025-09-05_22.11.58_1757592344181.png" alt="実装イメージ" />

注：ToDo リストの表示にも対応しています。本リポジトリには DingTalk カレンダー API を用いて予定情報を取得する実装例が含まれています。ほかの方法でデータを補完していただくことも可能です。

### 機能

-   カレンダー形式で現地の直近 30 日間の天気データを表示
-   当日の現地の天気、最低・最高気温を表示
-   当日の現地の気象警報を表示
-   現地の今後 2 時間内の雨・雪・霧の状況、または今後 2 週間内で最も近い該当日の情報を表示
-   ToDo リストを表示（デモを参考に DingTalk 予定を連携、または独自の ToDo テキスト取り込みを実装可能）

## 技術要件

ある程度の開発スキルと、各自のサーバ（またはローカル PC）が必要です。

実行環境：PHP + Apache/Nginx  
必要な PHP 拡張：curl, gd, mbstring
