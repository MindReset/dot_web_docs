## Quick Guide

Use the Device Settings API to read or update settings for a device owned by the authenticated API key user. Writes use `POST`; omitted fields stay unchanged.

### Read Settings

<ApiPlayground
	baseUrl="https://dot.mindreset.tech"
	endpoint="/api/authV2/open/device/:deviceId/settings"
	method="GET"
	pathParams={{
		deviceId: {
			type: 'string',
			required: true,
			name: 'Device serial number',
		},
	}}
	auth={true}
	defaultOpen={false}
/>

### Update Settings

<ApiPlayground
	baseUrl="https://dot.mindreset.tech"
	endpoint="/api/authV2/open/device/:deviceId/settings"
	method="POST"
	pathParams={{
		deviceId: {
			type: 'string',
			required: true,
			name: 'Device serial number',
		},
	}}
	body={{
		alias: 'Office Dot',
		location: 'Desk',
		timezone: 'Asia/Shanghai',
		interval: {
			powerMs: 300000,
			batteryMs: 10800000,
		},
		sleep: {
			enabled: true,
			start: '23:00',
			end: '07:00',
		},
	}}
	auth={true}
	defaultOpen={false}
/>

### Timezones

The `timezone` field only accepts keys returned by the Get Timezones API. See the Get Timezones page for the current server-supported timezone list.
