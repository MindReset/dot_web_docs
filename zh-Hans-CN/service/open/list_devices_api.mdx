## 问题排查

### 请求路径

`/api/authV2/open/devices`

### 请求方法

`GET`

### 请求结构

此接口不需要请求体,仅需在请求头中提供 API 密钥。

### 响应结构

<TypeTable
	type={{
		'[]': {
			description: '设备列表数组',
			type: 'array',
			required: true,
			items: {
				id: {
					description: '设备序列号',
					type: 'string',
					required: true,
				},
				series: {
					description: '设备系列（如 quote）',
					type: 'string',
					required: true,
				},
				model: {
					description: '设备型号（如 quote_0）',
					type: 'string',
					required: true,
				},
				edition: {
					description: '设备版本（1 或 2）',
					type: 'number',
					required: true,
				},
			},
		},
	}}
/>

### 响应说明

| code  | 含义       | 描述                 |
| ----- | ---------- | -------------------- |
| `200` | 成功       | 成功获取设备列表     |
| `401` | 未授权     | API 密钥无效或已过期 |
| `500` | 服务器错误 | 服务器内部错误       |
