## Clock

Clock displays date, time, and weekday on Rand/0.

-   Shows date, time, and weekday.
-   Supports Wi-Fi time sync.

## Controls

1. Long press the bottom button to enter.
2. If time has not been calibrated, the device will prompt you to enter Wi-Fi time sync mode.
3. Open **Settings** and choose **Wi-Fi Time Sync**.
4. After time sync is complete, return to **Clock**.
5. Long press the top button to return to the home screen.
