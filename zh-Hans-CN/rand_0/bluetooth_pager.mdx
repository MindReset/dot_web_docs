## 蓝牙遥控器

蓝牙遥控器可以将 Rand/0 连接到不同设备，用于翻页、远程拍照和演示控制。

-   连接大部分电子阅读器后，可作为翻页遥控器使用。
-   连接手机后，可作为远程拍照遥控器使用。
-   连接电脑后，可作为演示文稿遥控器使用，支持 Keynote、PowerPoint、Figma 等。
-   固件 1.2.0 及之后版本优化了 Apple 设备的蓝牙加密校验，提升连接兼容性。

## 按键模式

Rand/0 蓝牙遥控器支持多种按键输出模式。不同模式下，上键和下键会向已连接设备发送不同的蓝牙按键事件。

| 模式 | 适用场景 | 说明 |
| --- | --- | --- |
| 阅星曈协议 | 支持阅星曈协议的硬件设备 | 用于与支持该协议的硬件阅读设备配合使用。 |
| PPT 遥控 | Keynote、PowerPoint、Figma Slides 等演示场景 | 用于演示时上一页 / 下一页控制。 |
| 手机翻页 | 短视频、信息流、支持上下切换的手机 App | 用于在手机或平板上进行上下切换。 |
| 相机遥控 | 手机系统相机或支持音量键拍照的相机 App | 可作为远程快门使用。 |
| 媒体控制 | 音乐、视频或支持媒体键控制的 App | 用于上一曲 / 下一曲等媒体控制。 |
| 读书模式 | 微信读书、Apple 图书、电子阅读器等阅读场景 | 用于阅读翻页。 |
| 随想编码 | 支持 AI 快捷操作的工作流或 App | 默认上键为 **AI 接受**，下键为 **AI 召唤**。 |
| 自定义 | 需要自定义上下键行为的场景 | 将上键和下键分别设置为不同按键事件。 |

<Callout type="info" title="关于随想编码">
随想编码模式默认将上键设置为 **AI 接受**，下键设置为 **AI 召唤**。实际效果取决于连接设备或 App 是否支持对应按键事件。
</Callout>

### 自定义按键

在自定义模式中，上键和下键均可从以下按键事件中选择。

| 类别 | 可选按键事件 |
| --- | --- |
| 无操作 | 无 |
| 相机 / 音量 | 拍照 / 音量+、拍照 / 音量- |
| 媒体控制 | 播放 / 暂停、下一曲、上一曲 |
| 常用键盘 | 回车、空格、上页、下页、左箭头、右箭头 |
| 翻页器控制 | 翻页器-上页、翻页器-下页、翻页器-返回、翻页器-确认、翻页器-休眠、翻页器-关机、翻页器-关机前 |
| 演示控制 | F5 放映、Esc 退出、B 黑屏 |
| AI 控制 | AI 召唤、AI 接受、AI 内联 |

## 操作

1. 长按下键进入。
2. 等待设备提示 **等待链接**。
3. 在移动设备上打开蓝牙设置。
4. 找到 **Rand/0 Pager** 并配对。
5. 配对后根据设备提示继续操作。
