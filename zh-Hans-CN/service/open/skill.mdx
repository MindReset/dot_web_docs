## 什么是 Dot Skill？

Dot Skill 是一个开源工具，让 AI 助手能够通过自然语言指令直接与您的 Dot. 设备交互。无需手动编写 API 请求，只需告诉 AI 您想做什么。

## 功能特性

- **即插即用** - 安装后即可通过自然语言控制设备
- **文本推送** - 在设备上显示指定文本内容
- **图像推送** - 将图片推送到设备显示
- **画板推送** - 传入 data 和 windowData 绘制完整自定义画面
- **任务别名** - 为文本、图像、画板内容设置易识别的任务名称
- **状态查询** - 查看设备电量、网络状态等信息
- **内容切换** - 切换到下一个内容
- **辅助脚本** - 提供 Python 脚本用于自动化

## 支持平台

| 平台 | 状态 | 集成方式 |
| --- | --- | --- |
| Codex | 已支持 | 作为 Codex plugin 安装，使用仓库内 `.agents/plugins/marketplace.json` 与 `plugins/dot-skill` |
| OpenAI GPT Actions | 通过 schema 支持 | 导入 `openapi/dot-openapi.yaml` 并配置 Bearer 认证 |
| Claude / MCP clients | 计划中 | 目前可先使用 OpenAPI schema，后续可补 remote MCP server |
| Cursor 和兼容 skill 的 agent | 已支持为 skill 文档 | 安装 `skills/dot-device-openapi` 和/或 `skills/dot-canvas-designer`，或将本仓库作为上下文 |
| MCP Registry | 计划中 | 等 Dot MCP server 存在后发布 server metadata |

## 安装方式

### 方式一：作为 Codex 插件安装

将 dot_skill 仓库添加为 Codex marketplace：

```bash
codex plugin marketplace add git@github.com:MindReset/dot_skill.git
```

然后安装插件：

```bash
codex plugin add dot-skill@mindreset-dot-skill
```

安装完成后，请新建一个 Codex 线程，以便 Codex 加载插件中的 skills。

### 方式二：通过 `npx skills add` 安装

```bash
npx skills add https://github.com/MindReset/dot_skill.git
```

如果只需要设备交互能力：

```bash
npx skills add https://github.com/MindReset/dot_skill.git --skill dot-device-openapi
```

如果只需要 Canvas 设计能力：

```bash
npx skills add https://github.com/MindReset/dot_skill.git --skill dot-canvas-designer
```

### 方式三：手动安装

```bash
mkdir -p ~/.agents/skills
ln -sfn /path/to/dot_skill/skills/dot-device-openapi ~/.agents/skills/dot-device-openapi
ln -sfn /path/to/dot_skill/skills/dot-canvas-designer ~/.agents/skills/dot-canvas-designer
```

手动安装后请重启您的 agent。`dot-openapi` 仍作为旧安装的兼容入口保留，新安装建议使用 `dot-device-openapi` 与 `dot-canvas-designer`。

## 快速开始

### 配置环境变量

```bash
export DOT_API_KEY="dot_app_您的API密钥"
```

### 开始使用

安装完成后，您可以直接对 AI 助手说：

- "帮我在设备 ABCD1234ABCD 上显示 '会议在 3 点开始'"
- "帮我在设备 ABCD1234ABCD 上显示 '会议在 3 点开始'，并命名为会议提醒"
- "查看我的设备状态"
- "切换到下一个内容"

## 任务别名

如果您在同一台设备上创建了多个文本、图像或画板内容，可以为每个内容指定一个任务别名，方便在设备任务列表里区分用途。Dot Skill 会在推送文本、图像或画板时传递该名称；不指定时保留原有名称，明确要求清空名称时会移除别名。
