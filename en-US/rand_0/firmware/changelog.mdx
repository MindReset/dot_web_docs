<Timelines>

<Timeline>

### 1.2.5

- Wi-Fi time sync can now connect directly to remembered Wi-Fi networks for a smoother setup flow
- Fixed mismatches between Bluetooth remote labels and their actual functions, and fixed custom button saving issues
- Added a reading mode for Bluetooth remote control, with better support for apps such as WeChat Reading and Douyin desktop
- Fixed an issue where the Wooden Fish feature could leave unexpected screen ghosting in some cases
- Fixed repeated error loops that could occur when a custom wallpaper image was too large

</Timeline>

<Timeline>

### 1.2.3

- Improved device-wide power management for significantly better battery life
- Fixed an issue where personality guidance entries could display incorrectly

</Timeline>

<Timeline>

### 1.2.1

- Fixed text errors on the onboarding screens
- Added quick lock screen: long press the top button from the main menu to lock the screen
- Fixed an issue where Fortune content could switch multiple times in one day
- Added a reset personality option in Settings so you can choose your personality type again
- Added an option in Settings to reset the Wooden Fish counter
- Improved the screen refresh strategy: after every 9 partial refreshes, the device performs a full refresh to reduce ghosting
- Improved lock screen wallpaper display to reduce ghosting
- Added battery percentage display on the System Version page

</Timeline>

<Timeline>

### 1.2.0

- Improved Bluetooth encrypted verification for Apple devices for better connection compatibility

</Timeline>
