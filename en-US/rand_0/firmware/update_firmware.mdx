title: Update Device
description: Update Rand/0 firmware through Wi-Fi OTA.
---

## Update Device

1. Open **Settings** on Rand/0.
2. Select **Wi-Fi OTA**.
3. The device will use the last saved Wi-Fi network to perform the OTA update.

During the update, make sure the network connection is stable and do not power off the device.

## Use Another Wi-Fi Network

If you need to use another Wi-Fi network for **Wi-Fi OTA**, add that network first:

1. Open **Settings**.
2. Select **Wi-Fi Time Sync**.
3. Follow the on-device prompts to complete network setup.
4. After setup is complete, this Wi-Fi network can be used for **Wi-Fi OTA**.
