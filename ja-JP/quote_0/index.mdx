## Quote/0 とは

<CustomCloudImage osSrc="dot/docs/quote_0_hero.png" alt="Quote/0" />

Quote/0 は金属表面に吸着できるスマート電子ペーパー端末で、ミニマルなデザインを通じて情報過多を抑え、集中を促すことを目指しています。

アプリから表示コンテンツを柔軟に設定でき、RSS ニュース、ToDo、スケジュール、天気など多彩な情報に対応します。自分だけの表示戦略を組み、家族や友人とリモートで共有・管理・操作することも可能です。クラウドレンダリング技術により、豊かなレイアウトと多様な情報表現を実現。現在の内容に関心がある、または操作が必要な場合は、NFC 対応のスマートフォンを軽くタッチするだけで、iOS Clip App または Android アプリを起動して閲覧・操作できます。

コンテンツ工房は継続的に更新・拡張され、カスタムデザインプラットフォームやローコードシステムも今後提供予定です。

## 誕生ストーリー

Quote/0 の発想は、何気ないブレインストーミングから生まれました。しかし、その核には明確な感覚がありました——膨大で曖昧な情報を受け身で浴びるのではなく、日常空間の中に「主体的に選べる」コンテンツの入口を置くということ。

それは生活のどこかの一角に据えられ、続いていく個人的な展示のように存在します。見過ごすこともあれば、ある瞬間に凝視することもある。他のメディアに比べると、Quote/0 は不器用で、ゆっくりかもしれません。だからこそ、「無視」と「注視」の主導権をあなたに返し、出会いも見過ごしも、意識的に選び取る対話へと変えてくれます。

## Quote/0 の技術仕様

| 項目           | パラメータ説明          |
| :------------- | :---------------------- |
| 画面           | 2.66 インチ電子ペーパー |
| 解像度         | 296 × 152               |
| 画素密度       | 125 PPI/DPI             |
| バッテリー種別 | リチウム電池            |
| バッテリー容量 | 800 mAh                 |
| 正味重量       | 52 g                    |
| 総重量         | 102 g                   |

<Callout type="warn" title="注意事項">

-   端末にあらかじめ装着されているソフトマグネットを、他の磁力源や MagSafe に近づけないでください。磁性が意図せず失われる可能性があります。
-   電子ペーパー表面には強化プラスチックコーティングを使用しています。有機溶剤や腐食性の洗浄液で拭かないでください。
-   一部のスマホケースには磁石や金属プレートが含まれており、NFC 機能に影響する場合があります。使用前にケースを外してください。
-   出荷時のバッテリー残量は約 70% です。より長くご利用いただくため、満充電にしてから使用を開始してください。

</Callout>
