## Dot Calendar

開発者：wanyaxing | [GitHub リポジトリ](https://github.com/wanyaxing/dot_calendar)

今後 30 日間の天気を取得し、Quote/0 の電子ペーパーにプッシュ表示します。

<CustomCloudImage osSrc="dot/docs/iShot_2025-09-05_22.11.58_1757592344181.png" alt="実装イメージ" />

### 機能

-   カレンダー形式で地域の直近 30 日分の天気データを表示
-   当日の天気、最低・最高気温を表示
-   当日の気象警報情報を表示
-   向こう 2 時間の雨・雪・霧の予報、または今後 2 週間で最も近い日の雨雪霧を表示
-   ToDo リストを表示（デモのように DingTalk 予定に連携、または別途 ToDo テキスト連携を実装可能）

## Quote/0 MCP (stvlynn)

開発者：stvlynn | [GitHub リポジトリ](https://github.com/stvlynn/quote0-mcp)

Model Context Protocol（MCP）サーバーで、Dot. デバイスと対話するツール群を提供。Dot. 公式 API エンドポイントを実装し、デバイス画面にテキストや画像を表示します。

<CustomCloudImage osSrc="dot/docs/stvlynn_quote0-mcp_1757593061861.jpeg" alt="動作イメージ" />

### 機能

-   シンプルテキスト API：MindReset デバイスに基本的なテキストを表示
-   リッチテキスト API：署名やフォーマット付きのテキストを表示
-   アイコン付きテキスト：カスタムアイコンとタップでのリンク遷移を備えたテキスト表示
-   画像 API：カスタマイズ可能な枠線付き PNG 画像を表示

## DotClient

開発者：琥珀 | [GitHub リポジトリ / ダウンロード](https://github.com/onehupo/DotClient)

Dot. 向けに開発された電子ペーパー用プッシュソフト。独自のテキストや図案を作成・送信できます。

<CustomCloudImage osSrc="dot/docs/demo_1754558002770.jpg" alt="動作イメージ" />

### 主な機能

-   Dot. のテキスト・画像 API を用いてコンテンツを設定。画像は階調変換に対応し、白黒電子ペーパーでグレースケール表現
-   作図ページでテキストのスタイル（サイズ、回転、Emoji 等）を自由にカスタマイズ
-   設定した画像や作成した画像はファイルとしてエクスポートして共有可能
-   複数のサンプルアイコン・サンプル画像を同梱
-   複数デバイスを設定し、指定デバイスを選んで送信可能
-   base64 変換ツールを同梱し、画像を base64 に変換可能
