## Continuous updates...

We are committed to building an open ecosystem and will continuously explore innovative features based on community feedback. <Link href="/submit">Contribute your ideas or add your content to our Content Studio to reach more users</Link>.

### Planned

- Notes integration: daily note recap to make inspiration reviews easier.
- Docker Images Metadata: quickly fetch image metadata.
- Advanced API interactions: finer-grained device control (list devices, view status, reboot/sleep/switch content).

### Released

#### 2025.12.05

- GitHub User Activity: latest commits/PRs/issues for a user.
- GitHub Repo Data: key metrics like stars, branches, forks, and more.
- GitHub Pull Request: latest PRs for a selected repository.
- GitHub Issues: latest issues for a selected repository.
- Readwise Daily Review: daily highlight recap for steady learning.
- Gold Prices: daily gold price to watch the market.
- QR Code: display URL/text as a QR code.
- Number Board: show phone or any numeric info.
- Slot Machine: try your luck.
- PlayStation Monthly Games: monthly free games/updates.

#### 2025.9.25

- Telegram Channels: subscribe to channel updates; tap to view details.

#### 2025.9.10

- Photos: set your photos on Dot. devices to enjoy memories anytime.
- Weather Disaster Alerts: timely alerts to help keep travel safe.

#### 2025.9.03

- Rain Probability: next-hour rain chance and precipitation reminder.

#### 2025.8.8

- Calendar: lunar calendar and holiday info.
- Commuting Restrictions: alerts for common routes to avoid restrictions.
- Stock List: track up to four stocks with latest price and change.
- Real-time Weather: current conditions, feels-like, wind, and more at a glance.

#### 2025.7.17

- TickTick: sync a chosen list’s todos and manage status on-device.
- Text API / Image API: standard RESTful APIs for remote control or pushing custom content.

#### 2025.5.27

- YouTube: subscribe to channels and tap to view the latest video.
