創造所は「オープンかつ活発なコンテンツ集約プラットフォーム」で、ニュース、効率化、学習、移動、エンタメなど複数の分野をカバーしています。これにより、Quote/0 は単一コンテンツを表示するだけの電子ペーパー端末ではなく、生活に溶け込む小さなウィジェットとして無限の可能性を持つ存在になります。創造所は、これらを選択・編集するための入口です。

## 創造所の使い方

Dot. アプリではすでに創造所を公開しています。アプリのホーム画面で左へスワイプするか、下部の「創造所」をタップしてアクセスできます。
