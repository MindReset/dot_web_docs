title: Connect Device to Web Tool
---

<DeviceTool />

<Steps>
<Step>

### Use a data-capable cable to connect the device to your computer

Ensure the cable you use supports data transfer, not just charging. If no device appears in the pop-up window, it's usually because the cable only supports charging. We recommend using the original cable or another cable that supports data transfer.

</Step>

<Step>

### After ensuring a stable connection between the device and computer, click "Connect Device"

<Callout type="info">
	The web tool is compatible with Chromium-based browsers such as Arc, Chrome, Edge, and 360
	Browser. Do not use on mobile devices or in-app browsers.
</Callout>

<CustomCloudImage osSrc="dot/docs/device_tool_connect_1.jpeg" alt="Start connecting" />

</Step>

<Step>

### Select your device in the pop-up window

<CustomCloudImage osSrc="dot/docs/device_tool_connect_select.png" alt="Select device" />
<Callout type="info" title="If no device is available in the pop-up window">
    Please ensure the device is connected to the computer via data cable. If no device is still available, try changing the cable or the computer's port and try again.
</Callout>
</Step>

<Step>

### Connection complete

<CustomCloudImage osSrc="dot/docs/device_tool_connect_2.jpeg" alt="Connection complete" />

</Step>
</Steps>
