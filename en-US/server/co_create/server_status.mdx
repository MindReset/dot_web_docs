## Features

-   Real-time status collection: CPU, memory, disk, network, load, etc.
-   Circular progress visualization: ring bars for key metrics with high information density
-   Zero frontend dependency: Shell + HTTP API, ready to use out of the box
-   Scheduled push: supports BaoTa (BT Panel) cron jobs for periodic reporting
-   Quote integration: connects to the Quote/0 API; displays on the Quote panel and app
