コンテンツ工房は「オープンかつ活発なコンテンツ集約プラットフォーム」で、ニュース、効率化、学習、移動、エンタメなど複数の分野をカバーしています。これにより、Quote/0 は単一コンテンツを表示するだけの電子ペーパー端末ではなく、生活に溶け込む小さなウィジェットとして無限の可能性を持つ存在になります。コンテンツ工房は、これらを選択・編集するための入口です。

## コンテンツ工房の使い方

Dot. アプリではすでにコンテンツ工房を公開しています。アプリのホーム画面で左へスワイプするか、下部の「コンテンツ工房」をタップしてアクセスできます。
