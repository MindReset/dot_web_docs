一个为 Claude Code 和 Cursor 开发的 Model Context Protocol (MCP) 封装工具。该工具能够让 AI 编程助手在完成代码任务后，通过 MCP 协议在 Quote/0 墨水屏设备上自动显示任务完成通知。

适用于使用 Claude Code 或 Cursor 进行日常开发的用户，实现 AI 助手工作状态的实时可视化反馈。

### 使用场景

- 代码编写完成提醒
- 任务完成通知
- AI 工作状态展示

### 安装

```bash
npm install mindreset-dot-mcp
```
