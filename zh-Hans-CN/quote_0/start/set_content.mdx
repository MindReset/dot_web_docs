内容工坊是一个**开放且活跃的内容聚合平台**，已覆盖多个领域：资讯、效率、学习、出行、娱乐等。它让 Quote/0 不仅仅是一个显示单一内容墨水屏设备，而是一个拥有无限可能，融入你生活中的小组件，内容工坊则是选择和编辑这些的入口。

## 如何使用内容工坊

Dot. App 目前已开放内容工坊，在 Dot. App 主页中向左侧滑动，或点击下方「内容工坊」即可前往。
