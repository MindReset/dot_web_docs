## 什么是 Dot Skill？

Dot Skill 是一个开源工具，让 AI 助手能够通过自然语言指令直接与您的 Dot. 设备交互。无需手动编写 API 请求，只需告诉 AI 您想做什么。

## 支持的 AI 助手

- **GitHub Copilot** (VS Code / JetBrains)
- **Claude** (Claude Code / Claude Desktop)
- **ChatGPT** (通过自定义 GPTs)
- **其他支持 Skills/Agents 的 AI 工具**

## 功能特性

- 🔌 **即插即用** - 安装后即可通过自然语言控制设备
- 📝 **文本推送** - "帮我在设备上显示今天的待办事项"
- 🖼️ **图像推送** - "把这张图片推送到我的 Dot."
- 📊 **状态查询** - "我的设备电量还有多少？"
- 🔄 **内容切换** - "切换到下一个内容"
- 🐍 **辅助脚本** - 提供 Python 脚本用于自动化

## 快速开始

### 1. 获取 Skill

访问 GitHub 仓库获取最新版本：

👉 **[MindReset/dot_skill](https://github.com/MindReset/dot_skill)**

### 2. 安装 Skill

#### 方式一：通过 npx 安装（推荐）

```bash
npx skills add MindReset/dot_skill
```

#### 方式二：手动安装

```bash
mkdir -p ~/.agents/skills
ln -sfn /path/to/dot_skill/skills/dot-openapi ~/.agents/skills/dot-openapi
```

### 3. 配置环境变量

```bash
export DOT_API_KEY="dot_app_您的API密钥"
```

### 4. 开始使用

安装完成后，您可以直接对 AI 助手说：

- "帮我在设备 ABCD1234ABCD 上显示 '会议在 3 点开始'"
- "查看我的设备状态"
- "切换到下一个内容"
