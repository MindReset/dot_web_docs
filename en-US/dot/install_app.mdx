title: Install Dot. App
---

To activate and use Dot. series devices or features, please install the Dot. App first.

<App />

## Compatibility

- iOS: `iOS 17` and later
- Android: `Android 12L` and later
- HarmonyOS:
    - Versions `4.0` and below: Can be installed directly
    - Versions `HarmonyOS NEXT` and above: Can be installed using [Droitong](https://www.droitong.com)
