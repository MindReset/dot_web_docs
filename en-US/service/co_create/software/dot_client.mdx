A push client for Dot e-paper devices that lets you customize your own text and graphics.

### Key Features

-   Configure content via Dot text and image APIs; image grayscale conversion enables grayscale-like effects on black-and-white e-paper
-   Design page for freely customizing text styles, including size, rotation, and emoji
-   Export configured or created images for sharing
-   Multiple sample icons and selectable example images
-   Configure multiple devices and choose specific targets for sending
-   Built-in Base64 conversion tool to convert images to Base64
-   Exportable design-page configuration files for sharing
-   Supports partial macro-based substitutions
-   Supports automated tasks with macro definitions; recommended interval between tasks is over 5 minutes
