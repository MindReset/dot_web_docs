捷径 + 唯记 App + Quote/0 文字 API

「唯记（WAY 2 Reminder）」是一款专注于生日和纪念日提醒的 App，同样支持 Siri Intent 调用。借助捷径，你可以灵活获取倒数日信息并投放到墨水屏。

### 功能特性

- **生日和纪念日提醒**：专注于追踪重要日期和庆典的专业应用
- **Siri Intent 支持**：与 Siri 完全集成，支持语音查询
- **倒数日显示**：获取即将到来的事件倒数日信息
- **Quote/0 集成**：通过文字 API 在 Dot 设备的墨水屏上显示提醒倒数日
- **灵活配置**：通过捷径工作流自定义要显示的提醒内容
