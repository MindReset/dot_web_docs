## クイックガイド

タイムゾーン API では、現在サーバーが対応しているタイムゾーン一覧を取得できます。デバイス設定 API の `timezone` は、この API が返す `key` のみ指定できます。

<ApiPlayground
	baseUrl="https://dot.mindreset.tech"
	endpoint="/api/authV2/open/timezones"
	method="GET"
	auth={true}
	defaultOpen={false}
/>

### レスポンス例

```json
[
	{
		"key": "Asia/Shanghai",
		"name": "上海",
		"utcOffsetMinutes": 480,
		"utcOffsetLabel": "UTC+8"
	}
]
```
