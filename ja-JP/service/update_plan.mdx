## 継続的に更新中...

私たちはオープンなエコシステムの構築に取り組み、コミュニティからのフィードバックに基づいて、さらなる革新的機能の探索を続けます。<Link href="/submit">あなたのアイデアやコンテンツを創造所に投稿して、より多くのユーザーに届けましょう</Link>。

### 計画中

- メモ・ノート統合：毎日振り返りできるようにし、インスピレーション管理を容易に。
- Docker Images Metadata：イメージ情報を手軽に取得。

### リリース済み

#### 2025.12.19

- 開発者プラットフォーム API：デバイス一覧の取得、状態確認、次のコンテンツへ切替など、より細かなデバイス制御。<Link href="/docs/service/open">API ドキュメント入口</Link>。
- デイリー書籍引用：今日の日付に基づいて、ランダムに本の引用を表示します。

#### 2025.12.05

- GitHub ユーザー活動：最新のコミット/PR/Issue 動向を確認。
- GitHub リポジトリデータ：スター、ブランチ、フォークなどの指標。
- GitHub Pull Request：指定リポジトリの最新 PR。
- GitHub Issue：指定リポジトリの最新 Issue。
- Readwise デイリーリビュー：ハイライトを毎日振り返り。
- 天気予報：複数日の詳細予報。
- ゴールド価格：日々の金価格をチェック。
- QR コード：URL/テキストを QR 表示。
- ナンバープレート：電話など任意の番号を表示。
- スロット：運試し。
- Bluesky：指定ハンドルの最新投稿。
- 仮面ライダー公式ニュース：公式の最新アップデート。
- DLsite：新着/割引コンテンツ（種類選択可）。
- SEGA ニュース：公式アップデートや告知。
- 総務省消防庁：災害・消防関連のお知らせ。
- Yahoo ニュース：総合ニュース。
- NHK ニュース：公共放送の主要ヘッドライン。
- UNIQLO ニュース：ブランドの最新情報。
- PlayStation Monthly Games：毎月の無料ゲーム/更新。
- 日経新聞：ビジネス・マーケット主要ニュース。
- ロイター：グローバルニュース。
- ブルームバーグ：金融・マーケットニュース。
- 産経新聞：産経の主要ヘッドライン。
- 読売新聞：読売の主要ヘッドライン。
- 47NEWS 全国：全国ニュースダイジェスト。
- 47NEWS 地域：地域ニュースダイジェスト。
- 乗りものニュース：交通・乗り物系ニュース。
- ORICON NEWS：エンタメ/ランキング。
- まいどなニュース：総合ニュース。

#### 2025.9.25

- Telegram チャンネル：最新投稿を購読し、タップで詳細表示。

#### 2025.9.10

- 写真：Dot. デバイスに写真を設定し、いつでも思い出を楽しめます。
- 気象災害警報：タイムリーな警報情報で安全な外出をサポート。

#### 2025.9.03

- 降雨確率：1 時間先の降水確率と降雨量をお知らせ。

#### 2025.8.8

- カレンダー：旧暦と祝日情報を表示。
- 通勤・交通規制：よく使うルートの規制通知。
- 株価リスト：最大 4 銘柄の価格と騰落率。
- リアルタイム天気：現在の気温・体感・風向風力などを一目で。

#### 2025.7.17

- TickTick：指定リストの ToDo を同期し、デバイス側で管理。
- テキスト API / 画像 API：RESTful API で遠隔制御やカスタム表示を実現。

#### 2025.5.27

- YouTube：チャンネルを購読し、最新動画をタップで表示。
