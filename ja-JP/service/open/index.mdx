## 開発者プラットフォームとは？

開発者プラットフォームは、「自動化 / 連携」向けの機能セットです。主に次の要素で構成されます。

- **認証情報と認可**：Dot. アプリで API キーを作成・管理
- **デバイスとリソース**：デバイスのシリアル番号（デバイス ID）を取得し、対象デバイスを指定
- **公開 API**：標準的な HTTP リクエストで、コンテンツ操作・状態取得・デバイス一覧取得などを実行

まずは手早くデバイスへ内容を送ってみたい場合は、「クイックスタート」から始めるのがおすすめです。

### 利用のヒント（安全性と安定性）

- **API キーを安全に保管**：Dot. アプリは API キーを保存しません。漏えいの可能性がある場合は、アプリで削除して新しく作成してください。
