<Steps>
    <Step>
        ### 「少数派」ウェブサイトまたはアプリでログイン／登録

        1. [公式サイトへ](https://sspai.com)
        2. [アプリを入手](https://sspai.com/page/client)
    </Step>

    <Step>
        ### あなたの「少数派」リンクを取得

        <CustomCloudImage
            osSrc="dot/docs/evnydd0sf%20___%20-%20____1754392389016.jpeg"
            alt="正常なネットワーク"
            caption="公式サイトで右上のアバターをクリックし、「個人ホーム」を選択してURLをコピーします"
        />

        <div className="flex flex-col md:flex-row gap-4">
            <CustomCloudImage osSrc="dot/docs/STIIITCH_2025_08_05_07_19_43_1754392811893.PNG" alt="個人ホーム" caption="アプリでは、タブ「マイ」を開き、「個人ホーム」を選択し、右上の共有をタップ" />
            <CustomCloudImage osSrc="dot/docs/STIIITCH_2025_08_05_07_19_49_1754392830673.PNG" alt="共有ページ" caption="共有ページで「リンクをコピー」をタップ" />
        </div>
    </Step>
    <Step>
        ### 「少数派」の著者 ID を抽出

        取得したリンクが `https://sspai.com/u/evnydd0sf/updates` の形式の場合、`evnydd0sf` があなたの著者 ID です。
    </Step>
    <Step>
