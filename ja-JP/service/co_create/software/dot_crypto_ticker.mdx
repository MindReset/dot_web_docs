---
title: Dot 暗号通貨ティッカー
---

<GithubInfo owner="Linus-Shyu" repo="dot-crypto-ticker" />

Dot デバイスにリアルタイムの暗号通貨価格を表示する軽量な Rust アプリケーションです。デスクトップコンパニオンで市場データに即座にアクセスできます。

### 機能

- **マルチ暗号通貨対応**：ビットコイン (BTC)、イーサリアム (ETH)、テザー (USDT) の価格をリアルタイムで追跡
- **最適化された表示**：Dot デバイス画面専用に設計されたミニモードで、明確で読みやすいフォーマット
- **リアルタイム更新**：直感的なトレンド矢印 (↗/↘) 付きの 24 時間価格変動インジケーター
- **設定可能な間隔**：ニーズに合わせて更新間隔をカスタマイズ可能 (デフォルト：10 分)
- **安全な設定**：API キーとデバイス認証情報の環境変数ベースのセットアップ
- **24 時間 365 日バックグラウンド実行**：macOS バックグラウンドサービスとして継続的に動作
- **信頼性の高いデータ**：Binance API を使用して正確なリアルタイム暗号通貨市場データを提供

### 表示フォーマット

アプリケーションは Dot デバイス向けに最適化された、コンパクトで読みやすいフォーマットで暗号通貨価格を表示します：

```
BTC $111,266.93 ↗+2.44%
ETH $4,039.08 ↗+1.23%
USDT $0.9997 ↗+0.01%
```

各エントリには、桁区切り付きのフォーマットされた価格、方向指示器、24 時間のパーセンテージ変化が含まれます。
