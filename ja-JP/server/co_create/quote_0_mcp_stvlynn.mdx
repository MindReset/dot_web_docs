Dot デバイスとの連携ツールを提供する MCP (Model Context Protocol) サーバーです。Dot 公式の API エンドポイントを実装し、Dot デバイスの画面にテキストや画像を表示できます。

<CustomCloudImage osSrc="dot/docs/stvlynn_quote0-mcp_1757593061861.jpeg" alt="実行イメージ" />

### 機能

-   シンプルテキスト API: MindReset デバイスに基本的なテキストを表示
-   API: 署名や書式付きのテキストを表示
-   アイコン付きテキスト：カスタムアイコンとタップで遷移するリンク付きテキストを表示
-   画像 API: カスタマイズ可能な枠線付き PNG 画像を表示
