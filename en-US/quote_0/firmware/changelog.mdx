### 2.0.1

**Note:** After upgrading from 1.7.0 or below, you will need to tap "+" in the top-right of the App to re-add your device and reconfigure Wi-Fi. All previously configured content will be retained.

- Added OTA update support: the App can now view and update device firmware to the latest version
- Added multi-Wi-Fi support for saving multiple network configurations (App update soon)
- Added clock mode (App update soon)
- Improved Wi-Fi security with support for stronger encryption
- Improved connection stability and roaming in multi-router home networks
- Improved battery performance and battery level display accuracy
- Enhanced personal data encryption
- Optimized screen refresh speed for improved responsiveness
- Fixed display issue when network disconnects while charging
- Fixed occasional screen turning gray
- Fixed screen display abnormalities at low temperatures

</Timeline>
