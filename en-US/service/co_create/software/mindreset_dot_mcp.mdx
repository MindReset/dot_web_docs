A Model Context Protocol (MCP) wrapper designed for Claude Code and Cursor. This tool enables AI programming assistants to automatically display task completion notifications on Quote/0 e-ink devices through the MCP protocol.

Ideal for developers using Claude Code or Cursor in their daily workflow, providing real-time visual feedback of AI assistant work status.

### Use Cases

- Code completion reminders
- Task completion notifications
- AI work status display

```bash
npm install mindreset-dot-mcp
```
