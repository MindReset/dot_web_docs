## Dot Calendar

**开发者：wanyaxing** | [GitHub 仓库](https://github.com/wanyaxing/dot_calendar)

获取 30 天内天气并推送到 Quote/0 的电子墨水屏上。

<CustomCloudImage osSrc="dot/docs/iShot_2025-09-05_22.11.58_1757592344181.png" alt="实现效果" />

### 功能

-   使用日历的方式展示当地最近 30 天的天气数据
-   展示当天当地的天气、最低温度、最高温度
-   展示当天当地的预警信息
-   展示当地未来 2 小时内的雨雪雾天气情况，或未来两周内最近一日的雨雪雾。
-   展示 ToDo 列表（可参考 demo 接入钉钉日程，或另行实现相关 ToDo 文本的接入）

## Quote/0 MCP (stvlynn)

**开发者：stvlynn** | [GitHub 仓库](https://github.com/stvlynn/quote0-mcp)

一个模型上下文协议（MCP）服务器，提供与 Dot. 设备交互的工具。该服务器实现了 Dot. 官方 API 端点，用于在 Dot. 设备屏幕上显示文本和图像。

<CustomCloudImage osSrc="dot/docs/stvlynn_quote0-mcp_1757593061861.jpeg" alt="运行效果" />

### 功能

-   简单文本 API：在 MindReset 设备上显示基本文本内容
-   复杂文本 API：显示带有签名和格式的文本
-   带图标文本：显示带有自定义图标和点击跳转链接的文本内容
-   图片 API：显示带有可定制边框的 PNG 图像

## DotClient

**开发者：琥珀** | [GitHub 仓库 / 下载地址](https://github.com/onehupo/DotClient)

针对 Dot.开发的墨水屏推送软件，定制自己的文本和图案。

<CustomCloudImage osSrc="dot/docs/demo_1754558002770.jpg" alt="运行效果" />

### 主要功能

-   调用 Dot.的文本、图片 API 配置内容，图片支持灰阶转换，让黑白墨水屏显示灰度的效果
-   通过制图页面自由定制文本的样式，包括大小、旋转、Emoji 等
-   配置的图片或者制作的图片，可以导出文件分享
-   多个示例 icon，示例图片可以选择
-   配置多个设备，选择制定设备发送
-   添加了 base64 转换工具，可以把图片转换成 base64
