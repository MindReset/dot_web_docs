## 快速指引

获取时区 API 用于查询当前服务端支持的时区列表。设备设置 API 中的 `timezone` 只能使用此接口返回的 `key`。

<ApiPlayground
	baseUrl="https://dot.mindreset.tech"
	endpoint="/api/authV2/open/timezones"
	method="GET"
	auth={true}
	defaultOpen={false}
/>

### 响应示例

```json
[
	{
		"key": "Asia/Shanghai",
		"name": "上海",
		"utcOffsetMinutes": 480,
		"utcOffsetLabel": "UTC+8"
	}
]
```
