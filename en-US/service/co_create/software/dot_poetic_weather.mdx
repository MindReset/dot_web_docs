A PHP-based intelligent system that generates elegant poetry based on real-time weather information and pushes them directly to designated devices for display.

### Key Features

-   Real-time Weather Retrieval: Fetches accurate weather data via the QWeather (HeWeather) API
-   AI Poem Generation: Uses the GLM-4-Flash model to create poems that match the weatherâ€™s ambience
-   Device Push: Sends generated poem images directly to specified devices
-   Smart Layout: Optimized image layout with centered poetry and well-balanced information
-   Environment Configuration: Supports .env configuration for security and reliability
-   Multi-city Support: Accepts any city name and automatically resolves the city code
