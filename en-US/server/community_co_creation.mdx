## Dot Calendar

Developer: wanyaxing | [GitHub Repository](https://github.com/wanyaxing/dot_calendar)

Fetch 30 days of weather and push it to the Quote/0 e‑ink display.

<CustomCloudImage
	osSrc="dot/docs/iShot_2025-09-05_22.11.58_1757592344181.png"
	alt="Demonstration"
/>

### Features

-   Display local weather for the next 30 days in a calendar view
-   Show today’s local weather, minimum and maximum temperatures
-   Show today’s local weather alerts
-   Show local rain/snow/fog for the next 2 hours, or the nearest day with rain/snow/fog within the next two weeks
-   Show a ToDo list (refer to the demo for DingTalk schedule integration, or implement other ToDo text inputs)

## Quote/0 MCP (stvlynn)

Developer: stvlynn | [GitHub Repository](https://github.com/stvlynn/quote0-mcp)

An MCP (Model Context Protocol) server providing tools to interact with Dot. devices. It implements Dot.’s official API endpoints to display text and images on Dot. device screens.

<CustomCloudImage osSrc="dot/docs/stvlynn_quote0-mcp_1757593061861.jpeg" alt="Result" />

### Features

-   Simple Text API: display basic text on MindReset devices
-   Advanced Text API: display text with signatures and formatting
-   Text with icons: display text with custom icons and tappable links
-   Image API: display PNG images with customizable borders

## DotClient

Developer: Amber | [GitHub Repo / Download](https://github.com/onehupo/DotClient)

An e‑ink push app developed for Dot. to customize your own text and graphics.

<CustomCloudImage osSrc="dot/docs/demo_1754558002770.jpg" alt="Demonstration" />

### Key Features

-   Use Dot.’s text and image APIs to configure content; image grayscale conversion supported for richer black‑and‑white e‑ink rendering
-   Freely customize text styles on the drawing page, including size, rotation, and emoji
-   Export configured or created images for sharing
-   Includes multiple sample icons and images to choose from
-   Configure multiple devices and select specific targets to send
-   Built‑in base64 conversion tool to convert images to base64
