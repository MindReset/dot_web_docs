## Reset via App

<Callout type="info" title="When should I use 'Reset via App'?">
	When your device is connected to power and the current network is available. You will see the
	<LucideIcon icon={DeviceCurrentStatusTypeMap['IDLE_POWER'].icon} /> icon in the upper right corner of the
	device card in the Dot. App, indicating you can reset via the app.
</Callout>

## Network Tool

<Callout type="info" title="When should I use the “Network Tool”?">
	When your device cannot connect to the original network, indicating you need to use the web tool
	to reset the network.
</Callout>

### Next step

Tap “Reset Device” below. After the device screen changes, you can use the Dot. App to pair the device again.

<Callout type="info">
	If the screen does not change, try disconnecting, refreshing the page, and reconnecting before trying again.
</Callout>
