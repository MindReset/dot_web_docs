## NFC

口袋 NFC 是 Rand/0 口袋先知配套的 Android 小工具 App，用于 NFC 卡片相关能力。

下载地址：[Android 安装包](https://os-cdn.mindreset.tech/dot/device/rand_0/pocket-nfc-n1.2-c3_1777091076104.apk)

适用于 Android 设备。若浏览器拦截安装，请在下载完成后于系统通知或文件管理中打开 APK。

<Callout type="warn" title="兼容性说明">

-   口袋 NFC 目前仅提供 Android 安装包；
-   如果在微信内置浏览器中无法下载，请使用系统浏览器重新打开本页；
-   安装 APK 前，请根据系统提示允许来自当前浏览器或文件管理器的应用安装权限。

</Callout>

## 为什么不支持 iOS

iOS 的 NFC 功能未完全开放，因此口袋 NFC 无法在 iPhone 上完成实体 NFC 卡片的读取和写入操作。

你可以借用他人的 Android 设备完成 NFC 写入。写入完成后，Rand/0 可独立使用该 NFC 功能；只有在需要更换 NFC 内容时，才需要再次使用口袋 NFC App。

## 支持的卡片

Rand/0 内置无源 NFC 功能，可将本人拥有的个人实体 NFC 卡片写入设备中。该设备并不具有金融 IC 属性或能力。该功能仅适用于你本人拥有的 NFC 实体卡写入，请谨慎保管设备，切勿写入他人信息，也不要向他人共享安全卡片。

当前支持的卡片参数如下：

| 项目     | 参数说明                       |
| :------- | :----------------------------- |
| NFC Type | CUID                           |
| ATQA     | 0x0004                         |
| SAK      | 0x08                           |
| RAM      | 1kBytes (16 bytes Per Block)   |
