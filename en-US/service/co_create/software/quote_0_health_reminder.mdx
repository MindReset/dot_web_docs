## Features

Running this shortcut on iPhone will automatically retrieve the following data:

- Weight
- Body Fat Percentage
- Active Energy
- Steps
- Walking Distance

The data will be submitted to the server via API to automatically refresh the screen display.

## Technical Requirements

1. Know how to import and modify "Shortcuts" on iPhone
2. Be able to adjust the data sources for "Steps" and "Distance"
3. Be able to modify the authentication token and device SN in the final API submission interface

For more details, visit: [Health Reminder](https://ros6.com/?action=show&id=282)
