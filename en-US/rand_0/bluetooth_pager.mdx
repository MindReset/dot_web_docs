## Bluetooth Remote

Bluetooth Remote lets Rand/0 connect to different devices for page turning, remote photography, and presentation control.

-   Connect to most e-readers and use it as a page-turning remote.
-   Connect to a phone and use it as a remote camera shutter.
-   Connect to a computer and use it as a presentation remote for Keynote, PowerPoint, Figma, and more.
-   Firmware 1.2.0 and later versions improve Bluetooth encrypted verification for Apple devices for better connection compatibility.

## Button Modes

Rand/0 Bluetooth Remote supports multiple button output modes. In different modes, the up and down buttons send different Bluetooth button events to the connected device.

| Mode | Scenario | Description |
| --- | --- | --- |
| Yuexingtong protocol | Hardware devices that support the Yuexingtong protocol | Works with compatible hardware reading devices. |
| PPT Remote | Keynote, PowerPoint, Figma Slides, and other presentation scenarios | Controls previous / next slide during presentations. |
| Mobile Page Turning | Short-video apps, feeds, and mobile apps that support vertical switching | Switches content up and down on phones or tablets. |
| Camera Remote | System camera apps or camera apps that support volume-key shutter | Works as a remote shutter. |
| Media Control | Music, video, or apps that support media keys | Controls previous / next track and similar media actions. |
| Reading Mode | WeRead, Apple Books, e-readers, and other reading scenarios | Turns pages while reading. |
| Suixiang Coding | Workflows or apps that support AI shortcut actions | The default up button is **AI Accept**, and the default down button is **AI Summon**. |
| Custom | Scenarios that need custom up/down button behavior | Assign different button events to the up and down buttons. |

<Callout type="info" title="About Suixiang Coding">
In Suixiang Coding mode, the up button is set to **AI Accept** by default, and the down button is set to **AI Summon** by default. The actual effect depends on whether the connected device or app supports the corresponding button events.
</Callout>

### Custom Buttons

In Custom mode, both the up and down buttons can be assigned from the following button events.

| Category | Available button events |
| --- | --- |
| No action | None |
| Camera / Volume | Camera / Volume+, Camera / Volume- |
| Media control | Play / Pause, Next Track, Previous Track |
| Common keyboard | Enter, Space, Page Up, Page Down, Left Arrow, Right Arrow |
| Page-turner control | Page-turner Previous Page, Page-turner Next Page, Page-turner Back, Page-turner Confirm, Page-turner Sleep, Page-turner Power Off, Page-turner Before Power Off |
| Presentation control | F5 Present, Esc Exit, B Black Screen |
| AI control | AI Summon, AI Accept, AI Inline |

<Callout type="info" title="About Compatibility">
Actual Bluetooth Remote support depends on Rand/0 firmware, the connected device, and whether the corresponding app or hardware supports Bluetooth button events.

As apps, systems, or hardware firmware are updated, something that worked before may temporarily stop working; the opposite may also happen when a previously unsupported app or device starts supporting the relevant input.

If you notice a compatibility change, please leave us feedback. We will update this table and, when necessary, try to contact the corresponding app or hardware vendor to seek better compatibility support.
</Callout>

## Controls

1. Long press the bottom button to enter.
2. Wait until the device shows **Waiting for connection**.
3. Open Bluetooth settings on your mobile device.
4. Find **Rand/0 Pager** and pair it.
5. Follow the on-device prompts after pairing.
