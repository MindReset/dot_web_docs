---
title: 获取 API 密钥
description: 从 Dot. App 获取 API 密钥，以便在其他应用中使用 Dot. 的 API 功能。
---

<Callout type="info">
	<include>../../../dot/install_app.mdx</include>
</Callout>

<Steps>
  <Step>
    ### 在「更多」标签页，点击前往「API 密钥」

<CustomCloudImage
	osSrc="dot/docs/STIIITCH_2025_08_04_06_57_16_1754305764625.PNG"
	alt="前往「更多」标签页"
/>

  </Step>
    <Step>
    ### 点击「创建 API 密钥」

<CustomCloudImage
	osSrc="dot/docs/STIIITCH_2025_08_04_06_57_29_1754305976597.PNG"
	alt="创建 API 密钥"
/>

  </Step>
  <Step>
    ### 创建 API 密钥

    在创建 API 密钥后，请自行保管好密钥，Dot. App 不会保存您的 API 密钥。您需要**二次确认**已保存后才会关闭。

    <div className="flex flex-col md:flex-row gap-4">
    <CustomCloudImage osSrc="dot/docs/STIIITCH_2025_08_04_06_57_39_1754306146299.PNG" alt="创建 API 密钥 第一步" />
    <CustomCloudImage osSrc="dot/docs/STIIITCH_2025_08_04_06_57_45_1754306246880.PNG" alt="创建 API 密钥 第二步" />
    <CustomCloudImage osSrc="dot/docs/STIIITCH_2025_08_04_06_57_51_1754306318317.PNG" alt="创建 API 密钥 第三步" />

</div>

  </Step>
    <Step>
    ### 查看已创建的 API 密钥

    在此处您可以查看当前密钥创建的时间与最后使用的时间。

    <Callout type="warning" title="当您认为该密钥可能被泄露时，您可以随时删除该密钥。"/>

  </Step>
