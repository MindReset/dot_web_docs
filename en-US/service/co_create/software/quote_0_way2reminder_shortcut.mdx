Shortcut + WAY 2 Reminder App + Quote/0 Text API

"WAY 2 Reminder" (唯记) is an app focused on birthday and anniversary reminders, with full Siri Intent support. With Shortcuts, you can flexibly retrieve countdown information and display it on the e-ink screen.

### Features

- **Birthday & Anniversary Reminders**: Specialized app for tracking important dates and celebrations
- **Siri Intent Support**: Full integration with Siri for voice-activated queries
- **Countdown Display**: Retrieve countdown information for upcoming events
- **Quote/0 Integration**: Display reminder countdowns on your Dot device's e-ink screen via Text API
- **Flexible Configuration**: Customize which reminders to display through the shortcut workflow
