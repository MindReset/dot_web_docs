ショートカット + 唯記 App + Quote/0 テキスト API

「唯記（WAY 2 Reminder）」は、誕生日や記念日のリマインダーに特化したアプリで、Siri Intent にも対応しています。ショートカットを活用することで、カウントダウン情報を柔軟に取得し、電子ペーパーディスプレイに表示できます。

### 機能

- **誕生日と記念日のリマインダー**：重要な日付やお祝いを追跡する専門アプリ
- **Siri Intent サポート**：Siri との完全統合による音声クエリ対応
- **カウントダウン表示**：今後のイベントのカウントダウン情報を取得
- **Quote/0 統合**：テキスト API を介して Dot デバイスの電子ペーパー画面にリマインダーのカウントダウンを表示
- **柔軟な設定**：ショートカットワークフローで表示するリマインダーをカスタマイズ
