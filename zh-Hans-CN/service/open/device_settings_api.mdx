## 快速指引

设备设置 API 用于读取或更新一台归属于当前 API key 用户的设备设置。写入只支持 `POST`，未传入的字段会保持不变。

### 读取设置

<ApiPlayground
	baseUrl="https://dot.mindreset.tech"
	endpoint="/api/authV2/open/device/:deviceId/settings"
	method="GET"
	pathParams={{
		deviceId: {
			type: 'string',
			required: true,
			name: '设备序列号',
		},
	}}
	auth={true}
	defaultOpen={false}
/>

### 更新设置

<ApiPlayground
	baseUrl="https://dot.mindreset.tech"
	endpoint="/api/authV2/open/device/:deviceId/settings"
	method="POST"
	pathParams={{
		deviceId: {
			type: 'string',
			required: true,
			name: '设备序列号',
		},
	}}
	body={{
		alias: 'Office Dot',
		location: 'Desk',
		timezone: 'Asia/Shanghai',
		interval: {
			powerMs: 300000,
			batteryMs: 10800000,
		},
		sleep: {
			enabled: true,
			start: '23:00',
			end: '07:00',
		},
	}}
	auth={true}
	defaultOpen={false}
/>

### 时区

设备设置 API 的 `timezone` 字段只接受「获取时区」接口返回的 key。请先查看「获取时区」页面了解当前服务支持的时区列表。
