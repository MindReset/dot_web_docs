## Wi-Fi

After Rand/0 leaves the factory, all features that require internet access need Wi-Fi time sync to be completed once first. After this is done, the device remembers the connected Wi-Fi network and can continue using online features such as wallpaper changes.

## Wi-Fi Time Sync

When using Wi-Fi time sync on the device, follow these steps:

1. Scan the QR code shown on the device with your phone's **built-in system camera or QR scanner**;
2. After scanning, your phone system will automatically open the Wi-Fi connection page;
3. Connect to the corresponding Wi-Fi network on that page to complete Wi-Fi time sync.

Do not use a third-party QR scanner as the first choice. Some third-party scanner apps may not open the system Wi-Fi connection page correctly.

## Wi-Fi Memory

After Wi-Fi time sync is complete, the device automatically adds the connected Wi-Fi network to Wi-Fi Memory.

## Before Changing Wallpaper

Before changing the wallpaper for the first time, complete **Wi-Fi time sync** once. If Wi-Fi time sync has not been completed, features that require internet access may not work normally.
