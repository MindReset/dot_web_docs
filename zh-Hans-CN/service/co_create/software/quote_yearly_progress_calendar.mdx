一个基于 Quote/0 官方图像 API、苹果快捷指令与本地自动化的年度日历显示方案：每日推送一张 1-bit 风格的日历图片到设备，不进行高频刷新，直观呈现一年的进度。

<CustomCloudImage osSrc="dot/docs/IMG_8498_1758536679651.jpg" alt="实际效果" />

## 功能特性

-   每日自动更新 1 张日历图片（1-bit 风格）
-   顶部显示：日期、星期、电量百分比、当年天数索引
-   底部 365 个圆点：三种状态表示过去/当天/未来，直观呈现年度进度
-   可自定义 API Key 与设备 ID
-   支持本地自动化定时执行（无需手动）

## 实现思路

1.  生成资源
    -   使用 Python 批量生成 365 张日历图片（1-bit 风格）。
    -   动态元素（日期、星期、百分比等）拆分为独立素材，便于程序组合。
    -   将所有图片转换为 base64，并打包为一个 Json 文件。
2.  快捷指令侧
    -   读取 Json，按当天序号定位图片数据。
    -   通过“获取 URL 内容”模块调用 Quote/0 图像 API，推送到设备。
3.  展示
    -   上方：日期、星期、百分比、当天数。
    -   下方：365 圆点，三种状态显示年度进度。
