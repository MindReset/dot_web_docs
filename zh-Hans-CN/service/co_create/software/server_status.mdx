## 功能特性

-   实时状态采集：CPU、内存、磁盘、网络、负载等
-   环形进度可视化：关键指标环形条展示，信息密度高
-   零前端依赖：Shell + HTTP API，即开即用
-   定时推送：支持宝塔计划任务定时上报
-   Quote 集成：对接 Quote/0 API，展示于 Quote 面板与 App
