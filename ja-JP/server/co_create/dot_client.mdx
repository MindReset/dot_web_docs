Dot 向けに開発された電子ペーパー用プッシュソフトで、テキストや図案を自由にカスタマイズできます。

### 主な機能

-   Dot のテキスト/画像 API を用いてコンテンツを設定。画像はグレースケール変換に対応し、白黒電子ペーパーで階調表現を実現
-   作図ページでテキストのスタイルを自由にカスタマイズ可能 (サイズ、回転、emoji など)
-   設定済み画像や作成した画像をエクスポートして共有可能
-   複数のサンプルアイコンと選択可能なサンプル画像を同梱
-   複数デバイスを設定し、送信先デバイスを選択可能
-   Base64 変換ツールを搭載。画像を Base64 に変換可能
-   作図ページの設定ファイルをエクスポートして共有可能
-   一部のマクロ定義による置換をサポート
-   自動化タスクをサポート (同様にマクロに対応)。タスク間隔は 5 分以上を推奨
