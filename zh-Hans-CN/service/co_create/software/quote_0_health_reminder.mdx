## 功能特性

使用 iPhone 运行该快捷指令，将自动获取机主的：

- 体重
- 体脂率
- 活动能量
- 行走步数
- 行走距离

并使用 API 提交到服务器，实现屏幕数据自动刷新。

## 技术需求

1. 需要会操作苹果手机的"快捷指令"导入和修改
2. 能自己调整"步数"、"距离"数据来源
3. 能在最后提交 API 的接口里把认证的 token 和设备 SN 改为自己的
