Claude Code および Cursor 向けに開発された Model Context Protocol (MCP) ラッパーツールです。AI プログラミングアシスタントがコードタスクを完了した後、MCP プロトコルを通じて Quote/0 電子インクデバイスにタスク完了通知を自動表示します。

Claude Code または Cursor を日常的な開発に使用しているユーザーに最適で、AI アシスタントの作業状態をリアルタイムで視覚的にフィードバックします。

### 使用例

- コード完成リマインダー
- タスク完了通知
- AI 作業状態の表示

### インストール

```bash
npm install mindreset-dot-mcp
```
