## RSS とは？

RSS（Really Simple Syndication または Rich Site Summary）は、ウェブコンテンツのアグリゲーション形式で、ユーザーがサイトの更新情報を購読できる仕組みです。RSS を使うと、複数サイトの記事・ニュース・更新を一箇所でまとめて取得でき、個別にサイトへアクセスする必要がありません。

<Callout title="RSSHub について" type="info">

RSSHub は、RSS を提供していないサイトから RSS フィードを生成できるオープンソースの RSS 生成器です。RSSHub を用いることで、さまざまなサイトのコンテンツを RSS 形式へ変換し、統一的に購読・管理できます。Dot. のコンテンツ工房でデータの解析元が「RSSHub」と表示されている場合、そのデータは RSSHub の API を基に生成されています。

### 関連リンク

-   GitHub：[DIYgod/RSSHub](https://github.com/DIYgod/RSSHub)
-   公式ドキュメント：[docs.rsshub.app](https://docs.rsshub.app)

RSSHub を使えば、多様なウェブサイトのコンテンツを簡単に RSS 化し、真のコンテンツ集約購読を実現できます。

</Callout>
