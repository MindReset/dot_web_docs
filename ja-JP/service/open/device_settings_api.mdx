## クイックガイド

デバイス設定 API では、認証済み API key のユーザーが所有するデバイスの設定を取得または更新できます。書き込みは `POST` のみ対応し、省略したフィールドは変更されません。

### 設定を取得

<ApiPlayground
	baseUrl="https://dot.mindreset.tech"
	endpoint="/api/authV2/open/device/:deviceId/settings"
	method="GET"
	pathParams={{
		deviceId: {
			type: 'string',
			required: true,
			name: 'デバイスシリアル番号',
		},
	}}
	auth={true}
	defaultOpen={false}
/>

### 設定を更新

<ApiPlayground
	baseUrl="https://dot.mindreset.tech"
	endpoint="/api/authV2/open/device/:deviceId/settings"
	method="POST"
	pathParams={{
		deviceId: {
			type: 'string',
			required: true,
			name: 'デバイスシリアル番号',
		},
	}}
	body={{
		alias: 'Office Dot',
		location: 'Desk',
		timezone: 'Asia/Shanghai',
		interval: {
			powerMs: 300000,
			batteryMs: 10800000,
		},
		sleep: {
			enabled: true,
			start: '23:00',
			end: '07:00',
		},
	}}
	auth={true}
	defaultOpen={false}
/>

### タイムゾーン

`timezone` フィールドには、タイムゾーン取得 API が返す key のみ指定できます。現在サーバーが対応しているタイムゾーン一覧は「タイムゾーン取得」ページを確認してください。
