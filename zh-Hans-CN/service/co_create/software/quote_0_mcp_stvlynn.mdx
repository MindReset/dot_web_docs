一个模型上下文协议（MCP）服务器，提供与 Dot. 设备交互的工具。该服务器实现了 Dot. 官方 API 端点，用于在 Dot. 设备屏幕上显示文本和图像。

<CustomCloudImage osSrc="dot/docs/stvlynn_quote0-mcp_1757593061861.jpeg" alt="运行效果" />

### 功能

-   简单文本 API：在 MindReset 设备上显示基本文本内容
-   复杂文本 API：显示带有签名和格式的文本
-   带图标文本：显示带有自定义图标和点击跳转链接的文本内容
-   图片 API：显示带有可定制边框的 PNG 图像
