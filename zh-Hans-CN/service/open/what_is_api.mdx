---
title: 什么是 API
description: 快速了解 API 的概念及其工作原理，如果这是您第一次接触 API，从这里开始将再好不过。
---

API 是不同软件应用之间进行通信和数据交换的桥梁。简单来说，API 就像是一个"翻译官"或"服务员"，它帮助不同的程序互相"对话"。

## API 是如何工作的？

<Steps>
  <Step>
### 您发送请求

“请帮我在设备上显示‘今天天气很好’”

  </Step>
    <Step>
### API 处理请求

API 接收到请求后，会进行身份验证和设备验证，确保请求来自合法用户和设备。

  </Step>
    <Step>
### 设备执行操作

设备收到指令，在屏幕上显示文字

  </Step>
    <Step>
### 返回结果
"操作成功，内容已显示在设备上"

  </Step>
</Steps>

因此 Dot. API 让您能够：

- 当您收到重要邮件时，自动在设备上显示邮件标题和一部分内容
- 当您收取到验证码短信时，自动在设备上显示验证码
- 当门铃响起时，在设备上显示访客信息
- 呈现高度个性化的排版内容，例如个人化的信息仪表盘
- 显示有趣的图像或表情

## 谁可以使用开发者平台？

1. 极客爱好者：使用无代码工具（如 快捷指令、IFTTT、Zapier）创建简单的自动化；
2. 程序员：编写代码，创建复杂的自动化系统；
3. 企业用户：将 Dot. 集成到现有的业务系统中，提升工作效率；
4. 任何对物联网设备和自动化感兴趣的普通人。
