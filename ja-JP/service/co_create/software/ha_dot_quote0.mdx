MindReset の Dot. Quote/0 電子ペーパーデバイス向け Home Assistant カスタム統合です。Home Assistant からデバイスのステータスを監視し、テキストや画像コンテンツを Quote/0 に直接プッシュできます。

### 機能

#### センサー

- **電源状態** — 現在のデバイス電源モード（充電アクティブ、バッテリーアクティブ等）
- **バッテリー状態** — 充電状態
- **Wi-Fi 信号強度** — 信号強度（dBm）
- **ファームウェアバージョン** — 現在のファームウェア
- **最終レンダリング** — 最後の画面更新のタイムスタンプ
- **次回レンダリング（バッテリー/電源）** — 予定されている次回更新時刻

#### バイナリセンサー

- **オンライン** — 接続状態

#### コントロール（デバイスページ）

- **次のコンテンツ** — コンテンツループの次のアイテムに切り替え
- **テキスト送信** — タイトル・メッセージ・署名フィールドを使ってテキストをプッシュ
- **画像送信** — 画像データフィールドを使って画像をプッシュ
- **画像データ** — Base64 エンコードされた PNG（296×152px）または絶対ファイルパス
- **ディザリングタイプ** — ディザリングアルゴリズムを選択（DIFFUSION、ORDERED、NONE）

#### サービスアクション

- `dot_quote0.send_text` — 完全なパラメーター制御によるテキストコンテンツのプッシュ
- `dot_quote0.send_image` — ディザリングと枠線オプション付きの画像コンテンツのプッシュ

### インストール

#### HACS（推奨）

1. HACS → インテグレーション → ⋮ メニュー → カスタムリポジトリ を開く
2. `https://github.com/Never2land/HA-dot-quote0` をカテゴリ Integration で追加
3. Dot. Quote/0 をインストール
4. Home Assistant を再起動

#### 手動

`custom_components/dot_quote0/` フォルダーを Home Assistant の `config/custom_components/` ディレクトリにコピーして再起動します。

### 設定

1. 設定 → デバイスとサービス → 統合を追加 に移動
2. Dot. Quote/0 を検索
3. API キーを入力
4. すべての Quote/0 デバイスが自動的に検出されます
