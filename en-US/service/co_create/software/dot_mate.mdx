Dotmate is a scheduler for managing Quote/0 message push, supporting sending various types of messages to devices via cron-based tasks.

### Features

-   Scheduled tasks: a cron expression–based scheduling system
-   Multiple message types: supports text messages, work countdowns, code status, image messages, and title image generation
-   Multiple device management: manage multiple devices, each with its own task schedule
-   Flexible configuration: manage devices and tasks with YAML configuration files
-   Instant push: supports manually triggering message pushes
