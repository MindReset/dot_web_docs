A zero-backend, pure front-end, statically hostable web application that lets you send text content to Dot e-paper screens directly from the browser, with no app installation required.

## Core Features

-   Zero-backend: Implemented entirely on the front end, directly calling the official Dot API
-   Static hosting: Deployable to any static hosting service (Vercel, Netlify, GitHub Pages, etc.)
-   Internationalization: Built-in bilingual support (zh-CN / en-US)
-   Theme switching: Supports light/dark/system themes and follows system preference automatically
-   Privacy-first: API key is stored only within the browser session; not recorded or uploaded
-   Rate control: Automatically handles the 1 request/second API limit
-   Accessibility: Conforms to WCAG AA, with keyboard navigation and screen reader support
