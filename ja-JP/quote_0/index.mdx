## Quote/0 とは

<CustomCloudImage osSrc="dot/docs/quote_0_hero.png" alt="Quote/0" />

Quote/0 は金属面に取り付け可能なスマート電子ペーパー端末で、ミニマルなデザインにより情報過多を抑え、集中力を高めることを目指しています。

アプリから表示コンテンツを柔軟に設定でき、RSSニュース、ToDo、スケジュール、天気など多彩な情報に対応します。自分だけの表示戦略を組み立て、家族や友人とリモートで共有・管理・操作することも可能です。クラウドレンダリング技術により、豊かなレイアウトと多様な情報表現を実現しています。現在の内容に関心がある、または操作が必要な場合は、NFC対応のスマートフォンを軽くタッチするだけで、iOS Clip AppまたはAndroidアプリを起動して閲覧・操作できます。

コンテンツ工房は継続的に更新・拡張されており、カスタムデザインプラットフォームやローコードシステムも今後提供予定です。

## 誕生ストーリー

Quote/0 の発想は、何気ないブレーンストーミングから生まれました。しかし、その核には明確な感覚がありました。膨大で曖昧な情報を受け身で浴びるのではなく、日常空間の中に「主体的に選べる」コンテンツの入り口を置くということです。

それは生活のどこかの一角に据えられ、続いていく個人的な展示のように存在します。見過ごすこともあれば、ある瞬間に凝視することもあります。他のメディアに比べると、Quote/0 は不器用で、ゆっくりかもしれません。だからこそ、「無視」と「注視」の主導権をあなたに返し、出会いも見過ごしも、意識的に選び取る対話へと変えてくれます。

## Quote/0 の技術仕様

| 項目           | 仕様                    |
| :------------- | :---------------------- |
| 画面           | 2.66インチ電子ペーパー  |
| 解像度         | 296 × 152               |
| 画素密度       | 125 PPI/DPI             |
| バッテリー種別 | リチウムイオン電池      |
| バッテリー容量 | 800 mAh                 |
| 正味重量       | 52 g                    |
| 総重量         | 102 g                   |

<Callout type="warn" title="注意事項">

- 端末にあらかじめ装着されているソフトマグネットを、他の磁力源やMagSafeに近づけないでください。磁性が意図せず失われる可能性があります。
- 電子ペーパー表面には強化プラスチックコーティングを使用しています。有機溶剤や腐食性の洗浄液で拭かないでください。
- 一部のスマートフォンケースには磁石や金属プレートが含まれており、NFC機能に影響する場合があります。使用前にケースを外してください。
- 出荷時のバッテリー残量は約70%です。より長くご利用いただくため、満充電にしてから使用を開始してください。

</Callout>
