## 开发者平台是什么？

开发者平台是一组面向「自动化 / 集成」场景的能力集合，包含：

- **凭证与鉴权**：通过 Dot. App 创建并管理 API 密钥；
- **设备与资源**：获取并使用设备序列号（设备 ID）来区分不同设备；
- **开放 API**：通过标准 HTTP 请求控制设备内容、查询设备状态、获取设备列表等。

如果您只是想快速把内容推送到设备上，建议先从「快速开始」开始。

### 使用建议（安全与稳定）

- **请妥善保管 API 密钥**：Dot. App 不会为您保存密钥；如果您怀疑密钥泄露，可以随时在 App 中删除并重新创建。
