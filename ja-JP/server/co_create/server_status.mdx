## 機能

-   リアルタイム状態収集：CPU、メモリ、ディスク、ネットワーク、負荷など
-   リング型進捗可視化：主要指標をリングバーで表示し、高い情報密度を実現
-   フロントエンド依存ゼロ：Shell + HTTP API で、すぐに使い始められる
-   定期プッシュ：宝塔（BaoTa）パネルの cron タスクで定期報告に対応
-   Quote 連携：Quote/0 API に接続し、Quote のパネルおよびアプリに表示
