一个无服务器微服务，基于 AWS Lambda 和 DynamoDB，可将即将到来的日程事件显示在 Quote/0 提醒设备上，并集成了垃圾回收时间表的自动推送。

### 功能特性

- **定时推送**：每天 01:10、07:10、12:10、17:10 自动推送更新
- **垃圾回收集成**：对接 Reading Council API 并智能缓存数据
- **事件管理**：基于 DynamoDB 的自定义事件存储
- **Quote/0 适配**：遵循 25/29 字符限制的专属显示格式
- **无服务器架构**：AWS Lambda + DynamoDB，按需付费
- **自动弹性扩展**：自动应对流量峰值
- **低成本运营**：典型用量约 $1.18/月

### 架构

通过 EventBridge Cron 触发 Lambda，经由 API Gateway 提供以下接口：

- `GET /api/display` — 返回 Quote/0 设备所需的格式化展示数据
- `PUT /api/events` — 在 DynamoDB 中创建新事件

Quote/0 设备每小时主动拉取一次展示数据，同时支持来自 iPhone App 的事件写入。

### 技术栈

| 组件     | 技术                         |
| -------- | ---------------------------- |
| 运行时   | AWS Lambda (Node.js 18)      |
| 数据库   | Amazon DynamoDB              |
| 调度     | AWS EventBridge              |
| API      | API Gateway HTTP API         |
| 外部 API | Reading Council 垃圾回收 API |

### 定时推送时间

| 时间 (UTC) | 说明           |
| ---------- | -------------- |
| 01:10      | 凌晨刷新       |
| 07:10      | 工作日晨间更新 |
| 12:10      | 午间刷新       |
| 17:10      | 下班后晚间更新 |

可在 `serverless.yml` 中按所在时区调整。
