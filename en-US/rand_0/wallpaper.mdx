## Custom Wallpaper

1. Find **Settings** on Rand/0 and long press the bottom button to enter.
2. Find **Change Wallpaper** and long press to enter.
3. Scan the QR code with your phone.
4. Change the wallpaper on the opened page.
5. After the wallpaper is changed, the device will automatically return to the home screen.
6. Long press the top button from the main menu to quickly lock the screen and show the configured image. You can also open **Settings**, find **Lock Screen Now**, and long press to confirm.

Supported image formats: **jpg**, **jpeg**, and **png**. File size must be no larger than **5 MB**.

Changing wallpaper requires an available Wi-Fi Memory entry. If no Wi-Fi network has been added yet, complete Wi-Fi time sync from the **Wi-Fi** page first.

Firmware 1.2.1 and later versions improve lock screen wallpaper display to reduce ghosting.
