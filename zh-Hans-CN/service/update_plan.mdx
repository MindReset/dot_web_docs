### 滴答清单

_2025.7.17 已上线_

将您的滴答清单（非收集箱）待办事项无缝同步至 Dot.
设备。您可以在应用中连接滴答清单账户，即可开始使用。

<ul>
	<li>同步指定清单的待办事项</li>
	<li>在设备端管理任务状态</li>
</ul>

</Timeline>

### 文本 API / 图像 API

_2025.7.17 已上线_

我们为开发者提供了强大的 API 接口。通过 API 密钥构建标准请求，您可以实现对设备的远程控制，或将自定义内容推送至设备显示。

<ul>
	<li>提供标准 RESTful API 接口</li>
	<li>自定义内容推送与显示</li>
</ul>
